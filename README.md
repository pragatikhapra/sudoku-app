\# Sudoku App 🧩



\## 📌 Description

This is a Sudoku Solver and Generator web app with API support.



\## 🚀 How to Run



python3 -m venv .venv  

source .venv/bin/activate  

pip install -r requirements.txt  

python app.py  



Open:

http://127.0.0.1:5000/ui
